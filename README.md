## Цифровой учет для школьных библиотек
проект учащихся школы № 1862

# Запуск
- у вас должен быть установлен node js v18.16.1
1. скачайте архив, распакуйте и пререйдите в папку через косоль
2. выполните `npm i` чтобы установить зависимости
3. выполните `electron .` чтобы запустить

# сборка
- для сборки используется electron-packager:
`electron-packager .`
